# Amazon 価格追跡アプリ

AmazonのURLを登録すると毎日自動で価格をチェックし、設定した目標価格以下になると赤く表示するWebアプリです。

---

## 必要なもの

- **Python 3.10 以降**（未インストールの場合は下記を参照）
- インターネット接続

---

## セットアップ手順

### 1. Python のインストール（初回のみ）

1. https://www.python.org/downloads/ を開く
2. "Download Python 3.x.x" ボタンをクリック
3. インストーラーを実行し、**「Add Python to PATH」** にチェックを入れてインストール

### 2. アプリの起動

`start.bat` をダブルクリックするだけです。  
初回は自動でパッケージをインストールし、ブラウザが開きます。

---

## 使い方

| 操作 | 説明 |
|------|------|
| URL 入力 + 追加 | Amazon商品URLを貼り付けて「追加」ボタン |
| 目標価格 | 追跡したい価格を入力（省略可） |
| 🔴 赤表示 | 現在価格が目標価格以下のとき |
| 🔄 今すぐチェック | 手動で価格を更新 |
| 📋 価格履歴 | 過去の価格変動を確認 |
| ✏️ 目標価格 | 後から目標価格を変更 |
| 自動チェック | 毎日 **午前 9:00** に全商品を自動確認 |

---

## ファイル構成

```
Amazonの商品追跡/
├── start.bat          ← ダブルクリックで起動
├── server.py          ← Flask バックエンド
├── requirements.txt   ← 依存パッケージ
├── static/
│   └── index.html     ← フロントエンド UI
└── data/
    └── products.json  ← 商品データ（自動生成）
```

---

## 注意事項

- Amazonはスクレイピングをブロックする場合があります。その場合は少し時間をおいて再試行してください。
- アプリを終了するとスケジューラも停止します。自動チェックを継続するには `start.bat` を起動したままにしてください。
- データは `data/products.json` に保存されます。削除すると追跡データがリセットされます。
